# 🛢️ Strait of Hormuz Crisis — Supply Chain Impact Analyzer
> Built on live data during the active 2026 Iran-Hormuz oil disruption

## 📌 Project Overview
Real-world supply chain analytics project analyzing the cascading 
impact of the 2026 Strait of Hormuz closure on global oil prices, 
freight costs, and India's energy supply chain.

## 📊 Project Layers
| Layer | Title | Status |
|-------|-------|--------|
| 1 | Live Oil Price Tracker (WTI & Brent) | ✅ Done |
| 2 | Cascading Cost Impact Model | 🔄 In Progress |
| 3 | India Supply Chain Dashboard (Power BI) | ⬜ Coming |

## 🛠️ Tools Used
Python · Pandas · Matplotlib · FRED API · Power BI · Google Colab

## 📁 Data Sources
- FRED — Federal Reserve Economic Data (live API)
- IEA Emergency Report March 2026
- World Bank Trade Data
- PPAC India — Petroleum Planning & Analysis Cell
